# Portfolio Clean

転職活動向けのポートフォリオサイトです。  
トップページから 3 つの制作物（コーポレートサイト / 応募管理ミニアプリ / ITスクールLP）へ遷移できます。

## 公開URL

- Site: <https://williams7jpyu-dotcom.github.io/portfolio-clean/>

## 主な構成

- `index.html`: ポートフォリオ本体
- `works/cafe-latte.html`: 作品1（コーポレートサイト）
- `works/job-app-tracker.html`: 作品2（応募管理ミニアプリ）
- `works/it-school-lp.html`: 作品3（ITスクールLP）
- `works/detail-*.html`: 各作品の詳細解説ページ
- `assets/`: 各ページで使用する画像

## 補足

- 作品3のフォームはデモ実装です（実メール送信やDB保存は未実装）。
- 送信成功時は `works/it-school-lp-complete.html` に遷移します。
